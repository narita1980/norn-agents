# デモシナリオ — Microsoft Agent Hackathon 2026

動画撮影・Zenn 記事・ライブピッチ共通の **固定デモ手順** です。リハーサルはこの順番で 2 回以上行い、LLM 応答時間（合議 1–3 分）を見積もってください。

## 準備チェックリスト

- [ ] Azure デプロイ完了（[AZURE_DEPLOY.md](./AZURE_DEPLOY.md)）
- [ ] `NORN_APP_BASE_URL` が公開 URL に設定済み
- [ ] Azure OpenAI / GitHub Token が有効
- [ ] デモ用 GitHub リポジトリに **Draft PR** 用ブランチを用意
- [ ] Webhook 設定済み、または **手動 PR 登録** フォールバックを確認
- [ ] ブラウザ: シークレットウィンドウ 1 つ（審査員視点）、通常 1 つ（操作）
- [ ] GitHub PR ページを別タブで開いておく

## デモ用 PR の内容

意図的に「若手が学べる」小さな改善 PR を使います。

**推奨 diff テーマ**: 入力バリデーション不足 + エラーメッセージが分かりにくい

例（Python FastAPI 風）:

```python
# Before: バリデーションなし
@app.post("/items")
async def create_item(payload: dict):
    return {"id": 1, **payload}

# After 候補: Pydantic model + 400 レスポンス
```

**PR タイトル例**: `feat: add input validation for item creation`

**Draft PR** として open すること（Webhook トリガー条件）。

## シナリオ A — フルフロー（Webhook あり・推奨）

所要時間: **約 5 分**（合議待ち除く）

| Step | 操作 | 画面で見せるもの | ナレーション例 |
|------|------|------------------|----------------|
| 1 | GitHub で Draft PR を open | PR ページ（Draft バッジ） | 「若手が Draft PR を出すと、いきなり厳しいレビューが来る不安があります」 |
| 2 | Norn チャットを開く | 承認待ちバナー + PR 情報 | 「Norn は自動では動きません。若手が `[開始する]` を押すまで待機します」 |
| 3 | `[開始する]` をクリック | 右パネルに SSE 合議開始 | 「裏側ではウルド・ヴェルダンディ・スクルドが順に議論しています」 |
| 4 | 合議完了まで待つ | 3 女神 + モデレーターの発言 | 「技術・共感・成長の 3 視点が 1 本のレビューにまとまります」 |
| 5 | GitHub PR タブに切替 | Norn 投稿コメント | 「結果は GitHub にも届き、普段のワークフローのまま使えます」 |
| 6 | ダッシュボードタブ | KPI カード | 「シニア工数削減と若手学習時間を可視化します」 |

## シナリオ B — 手動 PR 登録（Webhook なし・フォールバック）

Webhook が使えない環境（録画スタジオ、審査会場）向け。

| Step | 操作 |
|------|------|
| 1 | チャット下部「手動 PR 登録」を開く |
| 2 | `owner/repo#123` 形式で PR URL を入力 |
| 3 | テストユーザー（ゆき）を選択したまま登録 |
| 4 | 以降はシナリオ A の Step 3 から同じ |

## シナリオ C — レベル別トーン差（任意・30 秒）

`LearnerSwitcher` で **ゆき（若手）→ さくら（上級）** を切替。

| 人物 | 同じ質問の例 | 期待される差 |
|------|-------------|-------------|
| ゆき | 「この PR の must_fix を教えて」 | 用語補足・小さな次の一手 |
| さくら | 同じ質問 | 簡潔・設計・保守視点 |

**PR 合議** でもスレッドの `user_level` がプロンプトに反映されます（手動登録時にレベルを指定）。

## 失敗時のリカバリ

| 失敗 | リカバリ |
|------|----------|
| Webhook が届かない | シナリオ B（手動登録）に切替 |
| 合議が 409 | 別スレッドを新規作成して再試行 |
| LLM タイムアウト | 事前録画クリップを差し込む（動画編集） |
| GitHub コメント失敗 | チャット上の合議結果を見せ「API 制限のため投稿スキップ」と説明 |
